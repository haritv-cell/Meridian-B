Connect your WHOOP.

You need a WHOOP developer app plus this dashboard deployed on Vercel. The Client Secret never touches the browser — it lives only in Vercel environment variables.
  1. Go to developer.whoop.com → create an app
  2. Set the app's redirect URL to your Vercel deployment: https://[YOUR-VERCEL-URL].vercel.app/api/whoop-callback
  3. Copy the Client ID (the secret stays in Vercel env vars only)
  4. In Vercel → Settings → Environment Variables, set WHOOP_CLIENT_ID, WHOOP_CLIENT_SECRET, and WHOOP_REDIRECT_URI
  5. Paste your Client ID below and click Save & connect
